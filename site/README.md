# site

This directory contains the public demo version of the dashboard.

It is meant to stay safe to commit:

- no private Spotify exports
- no live account credentials
- no browser session data

Run it locally with:

```powershell
cd D:\MYPROJECTS\yaMusicToSpotifyRelease\site
.\run_dashboard.ps1
```

The main working dashboard lives one level above in [`../dashboard`](../dashboard).
